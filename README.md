# Advanced_RAG

🎥 YouTube Video Summarizer & RAG Q&A Bot

An end-to-end Retrieval-Augmented Generation (RAG) application that extracts YouTube transcripts, generates concise summaries, and answers context-aware questions using semantic search and IBM Watsonx LLMs.

Built with LangChain, FAISS, IBM Watsonx Granite, and Gradio.

🧠 Tech Stack

  LLM Platform: IBM Watsonx
  
  LLM Model: Granite 3.2 8B Instruct
  
  Embeddings: IBM SLATE-30M English Retriever
  
  Vector Store: FAISS
  
  Framework: LangChain
  
  UI: Gradio
  
  Transcript Extraction: YouTubeTranscriptAPI
